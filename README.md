# yhttpd

a simple http server, for studing `HTTP` protocol, epoll and linux.

## Usage

```shell
$ ./yhttp -h

```
The default structure of `www` directory is shown as below.
```text
www/
|-- err-codes/
|   +-- 404.html    # etc. MUST
|
+-- index.html      # normal file
```
