yhttpd
======

A simple but powerful http server, for studying ``HTTP`` protocol, epoll and linux.

Usage
-----

.. code-block:: bash

  $ ./yhttp -h

The default structure of ``www`` directory is shown as below.

.. code-block:: text

  www/
  |-- err-codes/
  |   +-- 404.html    # etc. MUST
  |
  +-- index.html      # normal file
