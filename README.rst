======
yhttpd
======

A simple but powerful http server, for studying ``HTTP`` protocol, epoll and linux.

Usage
-----

.. code-block:: bash

  $ ./yhttp -h

The default configuration of ``yhttpd`` is dumped by ``./yhttpd -z``
